Run Directions: (requires anaconda)
1. Clone the repository
2. In terminal, navigate to directory that has the .yml file.
3. In the anaconda terminal, enter the command "conda env create -f quantumEnvironment.yml"
4. Again in the anaconda terminal, enter the command "conda activate <quantum>"
5. Navigate to \anaconda3\envs\quantum\Lib\site-packages\hybrid and paste the contents from decomposerHolder.py into decomposers.py in the hybrid directory.
6. Run join_order.py in this new environment.
